# Data Directory

This folder contains the structured scenario input datasets used for execution of the FES-MADM III framework.

## Included Files

- `S0_Baseline.xlsx`
- `S1_Inflation_Market_Tightening.xlsx`
- `S2_Cyber_Procurement_Disruption.xlsx`
- `S3_Market_Consolidation.xlsx`
- `S4_Budget_Tightening.xlsx`

## Format

Each file contains the scenario-specific multicriteria input matrix formatted for direct import into the provided R implementation.

## Notes

All datasets follow a harmonized structure to enable scenario comparison, ranking replication, and sensitivity analysis.
