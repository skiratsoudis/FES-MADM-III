# Results Directory

## Overview

This directory contains the consolidated analytical outputs generated through execution of the FES-MADM III framework across the available operating scenarios.

The files included in this folder are intended to support transparency, computational verification, scenario comparison, and independent review of the produced results.

---

## Main Output File

- `Full_Results.xlsx`

This workbook contains the structured outputs produced by the analytical platform after completion of the full computational workflow.

Depending on the executed configuration, outputs may include:

- Alternative rankings  
- Criterion weights  
- Normalized matrices  
- Aggregated performance scores  
- Scenario-by-scenario comparative results  
- Robustness indicators  
- Sensitivity-oriented summaries

---

## Purpose of the Results Package

The results repository serves multiple functions:

- Validation of reported findings  
- Reproduction of scenario outcomes  
- Cross-checking of rankings  
- Independent analytical inspection  
- Archival documentation of model execution

---

## Interpretation Notes

Results should be interpreted jointly with:

- the scenario datasets stored in `data/`
- the executable platform stored in `code/`

Because rankings and indicators may vary across scenarios, users are encouraged to compare outputs under different operating environments rather than relying on isolated single-case observations.

---

## Reproducibility

When the same scenario inputs and computational settings are used, the framework should reproduce equivalent outputs under the same software environment.

This enhances auditability and methodological transparency.

---

## Suggested Uses

The files contained in this directory may support:

- Academic replication studies  
- Comparative benchmarking exercises  
- Decision validation processes  
- Sensitivity analysis  
- Scenario planning exercises  
- Supplementary research documentation

---

## Final Remark

The results directory represents the observable analytical output of the FES-MADM III framework and complements the theoretical and computational components of the repository.
