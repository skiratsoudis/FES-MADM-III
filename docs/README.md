# Documentation Directory

## Overview

This directory contains supporting documentation associated with the FES-MADM III repository.

The materials included here are intended to improve interpretability, transparency, archival quality, and public usability of the computational package.

---

## Included Documents

- Methodological summaries  
- Release notes  
- Repository support files  
- Archival metadata

---

## Purpose

The documentation package complements the datasets, executable code, and analytical outputs provided in the repository.

It is designed to facilitate understanding of the framework without disclosing the full unpublished manuscript content.

---

## Final Remark

This directory supports professional dissemination, reproducibility standards, and long-term research archiving.
