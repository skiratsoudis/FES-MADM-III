# Release Notes

## Version 1.0.0

### Initial Public Reproducibility Release

This release represents the first structured public version of the FES-MADM III repository.

It includes the core computational assets required to inspect, execute, and reproduce the analytical workflow associated with the framework.

---

## Included Components

### Repository Structure

- `data/` scenario input datasets  
- `code/` executable analytical platform  
- `results/` consolidated outputs  
- `docs/` supporting documentation

### Scenario Package

The release includes the structured operating scenarios used for comparative execution:

- S0 Baseline  
- S1 Inflation / Market Tightening  
- S2 Cyber Procurement Disruption  
- S3 Market Consolidation  
- S4 Budget Tightening

### Computational Implementation

- R-based Decision Studio platform

### Analytical Outputs

- Consolidated results workbook containing generated outputs and rankings
